이미지 태그

속성 이름 설명
src 이미지나 오디오 경로 지정
alt 이미지가 없을 때 나오는 글
width 이미지의 너비 지정
height 이미지의 높이 지정

오디오 태그

속성 이름 설명
src 이미지나 오디오 경로 지정
preload 음악을 재생하기 전에 모두 블러올지 지정
autoplay 음악을 자동 재생할지 지정
loop 음악을 반복할지 지정
controls 음악재생 도구를 출력할지 지정

비디오 태그

속성 이름 설명
src 비디오 경로 지정
poster 비디오 준비 중일 때의 이미지 파일 경로 지정
preload 비디오를 재생하기 전에 모두 불러올지 지정
autoplay 비디오를 자동 재생할지 지정
loop 비디오를 반복할지 지정
controls 비디오를 재생 도구를 출력할지 지정
width 비디오의 너비 지정
height 비디오의 높이 지정
